Software made by me:

  • img2musicXML (GitHub) - Allows you to generate musicXML 2.0 files by drawing over a music score. Particularly useful when you want to listen what a score sounds like or to generate high-quality printable versions of scanned/photographed scores.
  • tf2_colored_message_generator (GitHub) - Easily generate `sm_me` colored messages for Team Fortress 2. Please not that only a few servers allow this.
  • Economagia (GitHub) - Android app for getting the lowest prices of commonly bought items in major stores and markets of São José dos Campos, SP, Brazil.
  • SiteHistory (GitHub) - Browser extension to easily see past versions of a website. Will be available for Chrome and Firefox.

Software in which I am interested:

  • ReactOS (GitHub) - A OS that plans to be fully Windows compatible someday. It's under active development since 1996 and can run old Windows software quite well.
  • Redox (GitHub) - An OS made from scratch and written in Rust that aims to be a general-purpose desktop workstation OS. It is POSIX inspired and compatible with some of its standards, although the author made it crystal clear that full POSIX support is not intended.

    In my opinion, if it attracts enough developers, it might be a replacement for the current linux desktop for many users, given it has its own DE and UI toolkit. Besides, it's very fast and not bloated (for now, at least).
